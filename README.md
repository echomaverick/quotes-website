# Quotes Website

This is a simple ReactJS web application that displays random quotes. Users can refresh the page to see different quotes, providing inspiration and motivation.

## Features

- **Random Quotes**: The website fetches random quotes from a public API and displays them to the user.
- **Refreshing**: Users can refresh the page to see a new random quote.
- **Responsive Design**: The website is designed to be responsive and accessible on various devices.

## Technologies Used

- **ReactJS**: The front-end of the website is built using ReactJS, a popular JavaScript library for building user interfaces.
- **Fetch API**: The Fetch API is used to fetch random quotes from a public API.
- **CSS**: Basic CSS styles are applied to enhance the appearance and layout of the website.

## Getting Started

To run the project locally, follow these steps:

1. Clone the repository:
  ```git clone (https://github.com/echomaverick/quotes-website)```

2. Navigate to the project directory:
  ```cd quotes-website```

3. Install dependencies:
  ```npm install```

4. Start the development server:
  ```npm start```

5. Open your browser and visit `http://localhost:3000` to view the website.

## API Used

This project uses the [Quotable Quotes API](https://api.quotable.io/random) to fetch random quotes. The API provides a collection of quotes that are suitable for use in various applications. But due to security reasons this api has been removed and will throw privacy error if you try to access it in the browser.
